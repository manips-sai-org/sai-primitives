MotionForceTask.h
1/*
2 * MotionForceTask.h
3 *
4 * This class creates a 6Dof position + orientation hybrid controller for a
5 * robotic manipulator using operational space formulation and an underlying PID
6 * compensator. If used for hybrid position force control, assumes a force
7 * sensor is attached to the same link as the control frame and the force sensed
8 * values are given in sensor frame. Besides, the force sensed and moment sensed
9 * are assumed to be the force and moment that the robot applies to the
10 * environment. It requires a robot model parsed from a urdf file to a SaiModel
11 * object, as well as the definition of a control frame as a link at which the
12 * frame is attached, and an affine transform that determines the position and
13 * orientation of the control frame in this link
14 *
15 * Author: Mikael Jorda
16 */
17
